**구현기능**

- 할일 추가 : 입력 후 + 버튼 클릭시 할일 추가
  - 빈칸으로 + 버튼 클릭시 'alert' 창 띄우기
- 할일 삭제 : 더블 클릭 시 리스트에서 해당 항목 삭제
- 할일 완료 : 한번 클릭 시 리스트의 해당 항목에 줄 긋기
